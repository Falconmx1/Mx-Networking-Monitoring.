# 🌐 Mx Networking Monitoring

Herramienta de monitoreo de red con interfaz gráfica, alertas en tiempo real y análisis de paquetes.

## 📦 Características
- Monitoreo de dispositivos en la red (ARP, ICMP)
- Análisis de tráfico en tiempo real
- Interfaz web amigable (Flask/Django)
- Alertas configurables por correo/telegram
- Reportes exportables en PDF/CSV

## 🛠️ Instalación
```bash
git clone https://github.com/Falconmx1/Mx-Networking-Monitoring.git
cd Mx-Networking-Monitoring
pip install -r requirements.txt
python main.py
